Структуры данных в Pandas: Series и DataFrame

 

Библиотека Pandas предоставляет две структуры: Series и DataFrame для быстрой и удобной работы с данными. Ранее использовалась еще одна структура - Panel, но в данный момент она имеет статус deprecated (устарело) и в будущем будет исключена из состава библиотеки Рandas.

Series - это маркированная одномерная структура данных, ее можно представить, как таблицу с одной строкой (или столбцом). С Series можно работать как с обычным массивом (обращаться по номеру индекса), и как с ассоциированным массивом (можно использовать ключ для доступа к элементам данных).

DataFrame - это двумерная маркированная структура. По сути, это обычная таблица, что выражается в способе ее создания и работе с ее элементами.

Panel - представляет собой трехмерную структуру данных. Рассматривать этот объект мы не будем, сосредоточимся на создании и получении доступа к элементам данных структур Series и DataFrame.

Структура данных Series

Для начала импортируем необходимые библиотеки:

>>> import pandas as pd

>>> import numpy as np

 Создать структуру Series можно на базе следующих типов данных:

  • словарь (dict) Python;
  • список (list) Python;
  • массив ndarray (из библиотеки numpy);
  • скалярная величина.

Конструктор класса Series выглядит следующим образом: Series(data=None, index=None, dtype=None, name=None, copy=False, fastpath=False)

Рассмотрим основные параметры конструктора:

  •  data — источник данных: массив, одно значение или dict (None по умолчанию); на базе этой структуры создается Series.
  •  index — одномерный список меток для элементов (None по умолчанию); длина должна совпадать с data.
  •  dtype — тип данных (numpy.dtype), по умолчанию определяется автоматически None.
  •  copy — если True, то создается копия массива данных (по умолчанию False). Обычно достаточно указать только data и index.

В большинстве случаев, при создании Series, используют только первые два параметра.

Создание Series из списка Python

Самый простой способ создать Series - это передать в качестве единственного параметра в конструктор список Python:

>>> s1 = pd.Series([1, 2, 3, 4, 5])

>>> s1

Результат выполнения:

0 1

1 2

2 3

3 4

4 5

dtype: int64

В этом примере Series построен на базе списка из языка Python. В левой колонке отображаются целочисленные индексы (начинающиеся с нуля), правая колонка содержит сами значения. Для доступа к элементам Series, в данном случае, можно использовать индексы элементов структуры.

Передадим в качестве второго элемента список строк (в нашем случае это отдельные символы). Это позволит обращаться к элементам структуры Series не только по численному индексу, но и по метке, это делает объект, похожим на словарь:

>>> s2 = pd.Series([1, 2, 3, 4, 5], ['a', 'b', 'c', 'd', 'e'])

>>> print(s2)

Результат выполнения:

a  1 

b 2

c 3

d 4

e  5

dtype: int64

Обратите внимание на левый столбец, в нем содержатся метки, которые мы передали в качестве index-параметра при создании структуры. Правый столбец - это по-прежнему элементы структуры.

Создание Series из ndarray массива numpy

Создадим массив ndarray из пяти чисел, аналогичный предыдущему списку. Предварительно импортируем библиотеки Рandas и Numpy:

>>> import numpy as np import pandas as pd

>>> ndarr = np.array([1, 2, 3, 4, 5])

>>> type(ndarr)   

Результат выполнения:

<class 'numpy.ndarray'="">

Теперь создадим Series с буквенными метками:

>>> s3 = pd.Series(ndarr, ['a', 'b', 'c', 'd', 'e'])

>>> print(s3)

Результат выполнения:

a  1 

b 2

c 3

d 4

e  5

dtype: int64

 

Как видим, Series, созданный из ndarray, аналогичен Series, созданному из списка Python.

Создание Series из словаря (dict)

Еще один способ создать структуру Series - это использовать словарь для одновременного задания меток и значений:

>>> d = {'a':1, 'b':2, 'c':3}

>>> s4 = pd.Series(d)

>>> print(s4) 

 Результат выполнения:

a  1 

b 2

c 3

d 4

e  5

dtype: int64

Ключи (keys) из словаря d станут метками структуры s4, а значения (values) словаря — значениями.

Создание Series с использованием константы

Рассмотрим следующий способ создания Series, при котором все элементы структуры принимают одно и то же значение (константу):

>>> a = 7

>>> s5 = pd.Series(a, ['a', 'b', 'c'])

>>> print(s5)

 Результат выполнения:

 a 7

 b 7 

 c 7 

 dtype: int64

В полученной структуре имеется три элемента с одинаковым содержанием.

Работа с элементами Series

Рассмотрим основные способы обращения к объекту Series в Рandas.

К элементам Series следует обращаться по численному индексу, при таком подходе работа со структурой не отличается от работы со списками в Python. Создадим примерную структуру  Series:

>>> s6 = pd.Series([1, 2, 3, 4, 5], ['a', 'b', 'c', 'd', 'e'])

>>> s6[2] 

Результат выполнения:

3

Также можно использовать обращение по метке, тогда работа с Series будет аналогична работе со словарем (dict) в Python:

>>> s6['d']

Результат выполнения:

4

Доступен синтаксис работы со срезами:

>>> s6[:2]

Результат выполнения:

a 1

b 2

dtype: int64

В поле для индекса возможно поместить условное выражение (Булева фильтрация):

>>> s6[s6 <= 3]

Результат выполнения:

a 1

b 2 

c 3

dtype: int64

Series поддерживают поэлементные арифметические операции: операции сложения, умножения вектора на число и т.п. применяются к соответствующим элементам:

>>> s7 = pd.Series([10, 20, 30, 40, 50], ['a', 'b', 'c', 'd', 'e'])

При сложении структур, их элементы складываются между собой:

>>> s6 + s7

Результат выполнения:

a 11

b 22

c 33

d 44

e 55

dtype: int64

При умножении структуры на число, все элементы структуры умножаются на данный множитель:

>>> s6 * 3

Результат выполнения:

a 3

b 6

c 9

d 12

e 15

dtype: int64

Структура данных DataFrame

Если Series представляет собой одномерную структуру, которую для себя можно представить, как таблицу с одной строкой (или одним столбцом), то DataFrame это уже двумерная структура - полноценная таблица с множеством строк и столбцов. Конструктор класса DataFrame выглядит так: DataFrame(data=None, index=None, columns=None, dtype=None, copy=False)

Основные параметры конструктора:

  • data (обязательный в большинстве случаев): исходные данные для заполнения DataFrame. Может принимать различные типы данных:
    • numpy.ndarray (массив структурированный или однородный)
    • Iterable (итерируемый объект)
    • dict (словарь, ключи которого становятся именами столбцов, а значения — данными столбцов)
    • другой DataFrame (pandas.DataFrame или pandas.Series)
    • список словарей, список списков или другие подобные структуры.

  • index (необязательный): метки для использования в качестве индекса (меток строк) результирующего DataFrame. Если не указан, по умолчанию используется RangeIndex (числовой индекс от 0 до N-1).
  • columns (необязательный): метки столбцов для использования в результирующем DataFrame. Если данные уже содержат метки столбцов (например, если data — это словарь), этот параметр выполняет выборку столбцов. Если не указан, по умолчанию используется RangeIndex.
  • dtype (необязательный): тип данных (dtype) для принудительного задания всем столбцам. Если None (по умолчанию), типы данных будут выведены автоматически на основе входных данных.
  • copy (необязательный): логическое значение (bool), по умолчанию None (в старых версиях False). Если True,

    то будет создана копия массива данных. Этот параметр в основном актуален, если data уже является объектом DataFrame или двумерным массивом ndarray. 

Рассмотрим на практике различные подходы к созданию DataFrame’ов.

Создание DataFrame из словаря

Для создания DataFrame используют словарь, элементами которого могут быть списки, структуры Series и т.д.

Рассмотрим пример, где элементы словаря — это структуры Series:

>>> d = {'price':pd.Series([1, 2, 3], index=['v1', 'v2', 'v3']),'count': pd.Series([10, 12, 7], index=['v1', 'v2', 'v3'])}

>>> df1 = pd.DataFrame(d)

>>> print(df1)

     price  count

v1    1      10

v2    2      12

v3    3       7

Следующей командой выводятся на экран индексы созданного DataFrame в следующем виде:

>>> print(df1.index) Index(['v1', 'v2', 'v3'], dtype='object') 

 

Столбцы созданного DataFrame выводятся командой:

>>> print(df1.columns) Index(['price', 'count'], dtype='object')

Построим аналогичный словарь, но используем элементы массива ndarray:

>>> d2 = {'price':np.array([1, 2, 3]), 'count': np.array([10, 12, 7])}

>>> df2 = pd.DataFrame(d2, index=['v1', 'v2', 'v3'])

>>> print(df2)

Результата выполнения:

price count

v1 1 10

v2 2 12

v3 3 7

Индексы и столбцы второго DataFrame выводятся :

>>> print(df2.index) Index(['v1', 'v2', 'v3'], dtype='object')

>>> print(df2.columns) Index(['price', 'count'], dtype='object')

Как видно - результат аналогичен предыдущему. Таким образом, можно применять обычные списки Python вместо ndarray.

Создание DataFrame из списка словарей

Создадим DataFrame из списка, элементами которого являются словари:

>>> d3 = [{'price': 3, 'count':8}, {'price': 4, 'count': 11}]

>>> df3 = pd.DataFrame(d3)

>>> print(df3)

Результат выполнения:

count price

0 8 3

1 11 4

Для получения сводной информации о созданном DataFrame применяется функция info(). Она отображает сведения о типе структуры, количестве записей, количестве non-null элементов в столбцах, типы и количество хранимых элементов и объеме используемой памяти:

>>> print(df3.info())

Результат выполнения:

<class 'pandas.core.frame.dataframe'="">

RangeIndex: 2 entries, 0 to 1

Data columns (total 2 columns):

count 2 non-null int64

price 2 non-null int64

dtypes: int64(2)

memory usage: 112.0 bytes

None

Создание DataFrame из двумерного массива

DataFrame создается также из двумерного массива, в данном случае это ndarray из библиотеки Numpy:

>>> nda1 = np.array([[1, 2, 3], [10, 20, 30]])

>>> df4 = pd.DataFrame(nda1)

>>> print(df4)

Результат выполнения:

  0  1    2

0   1  2   3

1 10 20 30

Работа с элементами DataFrame

Рассмотрим наиболее часто используемые подходы для доступа к элементам структуры. Основные способы представлены в таблице:

Операция      Синтаксис     Возвращаемый результат
Выбор столбца df[col] Series
Выбор строки по метке df.loc[label] Series
Выбор строки по индексу df.iloc[loc] Series
Срез по строкам df[0:4] DataFrame
Выбор строк, отвечающих условию df[bool_vec] DataFrame

Рассмотрим работу с данными операциями на практике. Для начала создадим DataFrame:

>>> d = {'price':np.array([1, 2, 3]), 'count': np.array([10, 20, 30])}

>>> df = pd.DataFrame(d, index=['a', 'b', 'c'])

>>> print(df)

Результат выполнения:

price count a b c 1 10 2 20 3 30

Операция выборки отдельных столбцов из таблицы данных осуществляется следующим образом:

>>> df['count']

Результат выполнения:

a 10

b 20

c 30

Name: count, dtype: int32

Операция извлечения строки по её индексной метке:

>>> df.loc['a']

Результат выполнения:

price 1

count 10

Name: a, dtype: int32

Операция выбора строки по индексу:

>>> df.iloc[1]

Результат выполнения:

price 2

count 20

Name: b, dtype: int32

Операция создания среза по строкам:

>>> df[0:2]

Результат выполнения:

price count

a 1 10

b 2 20

Операция отбора строк, отвечающих заданному условию:

>>> df[df['count'] >= 20]

Результат выполнения:

price count

b 2 20

c 3 30
